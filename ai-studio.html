Quick Start

# 1. Install requirements
pip install pandas numpy scikit-learn openpyxl joblib xgboost

# 2. Save the agent code as openmind_agent.py

# 3. Create a Python script:
from openmind_agent import OpenMindAgent
import pandas as pd

# Load your data
data = pd.read_csv("your_data.csv")

# Create and train agent
agent = OpenMindAgent(model_type="classification", algorithm="random_forest")
agent.train(data, target_col="target_column")

# Make predictions
prediction = agent.predict({"feature1": 10, "feature2": 20})
print(f"Prediction: {prediction}")

Basic Usage

1. Load your data:

data = agent.load_data("sales_data.csv")  # CSV
data = agent.load_data("sales_data.xlsx") # Excel
data = agent.load_data("sales_data.json") # JSON

2. Train the model:

agent.train(data, target_col="price")  # For regression
agent.train(data, target_col="purchased")  # For classification

3. Make predictions:

# Single prediction
pred = agent.predict({"sqft": 2000, "bedrooms": 3})

# Batch prediction
new_data = pd.read_csv("new_customers.csv")
predictions = agent.predict(new_data)

Complete Examples

📊 Classification Example

from openmind_agent import OpenMindAgent
import pandas as pd

# Sample customer data
data = pd.DataFrame({
    'age': [25, 30, 35, 40, 45],
    'income': [50000, 60000, 70000, 80000, 90000],
    'purchased': [0, 0, 1, 1, 1]  # Target: will they purchase?
})

# Train agent
agent = OpenMindAgent(model_type="classification", algorithm="random_forest")
agent.train(data, target_col="purchased")

# Predict for new customers
new_customer = {'age': 32, 'income': 65000}
prediction, confidence = agent.predict(new_customer)
print(f"Will purchase? {prediction} (confidence: {confidence:.1%})")

# Save model
agent.save("purchase_predictor.pkl")

📈 Regression Example

from openmind_agent import OpenMindAgent
import pandas as pd

# House price data
data = pd.DataFrame({
    'sqft': [1500, 1800, 2200, 2500, 3000],
    'bedrooms': [3, 3, 4, 4, 5],
    'price': [250000, 300000, 350000, 400000, 500000]  # Target
})

# Train regression agent
agent = OpenMindAgent(model_type="regression", algorithm="random_forest")
agent.train(data, target_col="price")

# Predict house price
new_house = {'sqft': 2000, 'bedrooms': 3}
predicted_price = agent.predict(new_house)
print(f"Estimated price: ${predicted_price:,.0f}")

💾 Load Saved Model

# Load previously saved model
agent = OpenMindAgent()
agent.load("purchase_predictor.pkl")

# Use it for predictions
prediction = agent.predict({'age': 28, 'income': 55000})
print(f"Prediction: {prediction}")

Troubleshooting

• ModuleNotFoundError – Install missing packages: pip install [package_name]

• KeyError: 'target_column' – Check your column name, it must exist in your data

• ValueError: could not convert string to float – Encode categorical variables or use preprocess_data()

• ImportError: XGBoost not installed – Install XGBoost or use another algorithm
